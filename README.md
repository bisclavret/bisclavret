### Prerequisites

- Node.js 18+ 
- Rust 1.70+

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bisclavret/bisclavret.git
   cd bisclavret
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run in development mode**
   ```bash
   npm run tauri dev
   ```

### Building

```bash
# Build for production
npm run build

# Build Tauri app
npm run tauri build
```
